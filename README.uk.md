### 🛡️ Міні-сканер вразливостей веб-додатків

![Версія для Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Ліцензія: GPL v3](https://img.shields.io/badge/license-GPLv3-blue.svg)
![Статус](https://img.shields.io/badge/status-active-success.svg)

Простий, швидкий та ефективний інструмент Python CLI для базового аудиту безпеки веб-додатків. Призначений для розробників, які хочуть швидко перевірити свої сайти на наявність типових помилок конфігурації.

### 🚀 Функції (версія 2.0)

- [x] - **Аналіз заголовків безпеки:** перевіряє наявність критичних заголовків HTTP (HSTS, CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy тощо).
- [x] - **Брутфорс каталогів:** Активно сканує на наявність відкритих прихованих файлів та каталогів (наприклад, `/.env`, `/.git/`, `backup.zip`, `wp-config.php`).
- [x] - **Генерація звітів:** Експортуйте результати сканування у структурований формат `.json` або зручний для читання формат `.txt` для подальшого аналізу.
- [x] - **Захист від обходу ботами:** Використовує рандомізовані заголовки `User-Agent` для обходу базових WAF та обмежень сервера.
- [x] - **Інтерактивний інтерфейс терміналу:** Зручний вивід з кольоровим кодуванням (`colorama`) та стильним індикатором прогресу в реальному часі (`tqdm`).

### 🛠️ Встановлення та налаштування

1. Клонуйте репозиторій:
   ```bash
   git clone https://github.com/y-lebid/mini-web-scanner.git
   cd mini-web-scanner

2. Встановіть залежності
   ```bash
   pip install requests colorama tqdm
   ```
3. Запустіть
   ```bash
   python scanner.py
   ```
